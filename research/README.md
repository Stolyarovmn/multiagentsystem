# research/

Source reviews for external repositories, articles, and patterns considered for adoption into `multiagentsystem`.

## When to create a review

Create a source review when:
- you encounter an external repository with patterns worth evaluating
- a proposal issue references an external source
- you want to formally document why an external idea was rejected

## How to create a review

1. Copy `SOURCE_REVIEW_TEMPLATE.md`
2. Name it `SOURCE-NNN-short-name.md` (e.g. `SOURCE-001-oh-my-claudecode.md`)
3. Fill in all sections
4. Reference it from the related proposal issue
