# Tasks

Local task files live as `tasks/TASK-*.md` and are ignored by git by default.

Each task must include:

- `STATUS`
- `CREATED`
- `AUTHOR`
- `TARGET`
- `Goal`
- `Criteria`
- `Context`
- `Dispatch Envelope`
- `Attempt Log`
- `Audit Notes`

Statuses:

- `QUEUED`
- `IN_PROGRESS`
- `REVIEW`
- `DONE`
- `BLOCKED`

If a task becomes important framework history, promote it to a GitHub issue or an explicit docs artifact.
