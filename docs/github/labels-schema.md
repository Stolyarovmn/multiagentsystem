# Labels Schema

Recommended labels:

- `type:task`
- `type:dispute`
- `type:process-change`
- `type:blocker`
- `status:queued`
- `status:in-progress`
- `status:review`
- `status:done`
- `status:blocked`
- `role:orchestrator`
- `role:executor`
- `role:auditor`

Labels support scanning and dashboards. Project fields or issue state remain the stronger lifecycle signal when configured.
