# Project Fields

Recommended GitHub Project fields for framework development:

- `Status`: `QUEUED`, `IN_PROGRESS`, `REVIEW`, `DONE`, `BLOCKED`
- `Role owner`: `orchestrator`, `executor`, `auditor`, `human`
- `Retry count`: number
- `Needs audit`: boolean
- `Dispute linked`: issue reference
- `Target`: text

GitHub Projects manage framework work. Local runtime `STATE.md` manages live local execution.
